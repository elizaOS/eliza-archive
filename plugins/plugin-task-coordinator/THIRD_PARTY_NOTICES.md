## odysseus

The `src/odysseus/` chat-shell UI is an independent React reimplementation on
the `@elizaos/ui` stack, modeled on the UI/UX design of the **odysseus**
project. None of odysseus's Python backend is used — only the front-end design
was a reference. odysseus is distributed under the MIT License.
